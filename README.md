# Money Bot
I can exchange $, €, ₽ currencies. Ask me '$4' for example. Or '100 ₽'.

Telegram chat bot like
* you ask `$10`
* she replies `735 ₽`

# Installation

```sh
$ bundle
$ echo "TOKEN = 'YOUR_TOKEN'" >> token.rb
$ echo "OXR_APP_ID = 'Open Exchange Rates token'" >> token.rb
$ bundle exec ruby bot.rb
```
